# AI Context & Documentation

This directory contains the "Second Brain" of the project, designed to be ingested by AI agents (Cursor, Antigravity) to understand the full context of Summagram.

## Key Files

- **[SYSTEM_OVERVIEW.md](./SYSTEM_OVERVIEW.md)**: The map of the territory. High-level goals, core entities, and module descriptions.
- **[TECH_STACK.md](./TECH_STACK.md)**: The rules of engagement. Languages, libraries, and strict coding conventions.
- **[ADR_LOG.md](./ADR_LOG.md)**: Architecture Decision Records. The "Why" behind our technical choices.

## Usage for AI
When starting a new task, always read `SYSTEM_OVERVIEW.md` and `TECH_STACK.md` first. Consult `ADR_LOG.md` before proposing architectural changes.
