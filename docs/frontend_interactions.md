# Взаимодействия с Фронтендом

В этом документе описаны все интерактивные элементы интерфейса Summagram, возможные значения полей и соответствующие API-зазовы.

## Общие навигационные элементы (Sidebar)

| Элемент | Тип | Действие | API Вызов |
| :--- | :--- | :--- | :--- |
| **New Session** | Кнопка | Создает новую пустую сессию и переключает на вид чата | `POST /api/sessions` |
| **Chats** | Ссылка | Переход к выбору чата из Telegram | `GET /api/chats` |
| **Sessions** | Ссылка | Переход к списку существующих AI-сессий | `GET /api/sessions` |
| **Datasets** | Ссылка | Просмотр всех проиндексированных документов | `GET /api/documents` |
| **Pipelines** | Ссылка | Настройка параметров обработки данных | `GET /api/config` |
| **Network** | Ссылка | Просмотр социального графа | `GET /etl/graph/data` |

---

## 1. Выбор чата (Chats / Session Select)

| Элемент | Тип | Описание / Данные | API Вызов |
| :--- | :--- | :--- | :--- |
| **Карточка чата** | Кнопка | Клик по чату создает новую сессию на основе этого чата | `POST /api/sessions` (с `context_chat_id`) |
| **Filter Chats** | Collapsible | (В разработке) Будет содержать фильтры по названию и дате | - |

---

## 2. Чат с AI (Chat View)

| Элемент | Тип | Описание / Данные | API Вызов |
| :--- | :--- | :--- | :--- |
| **Поле ввода** | Input | Текст сообщения пользователя | - |
| **Кнопка Send** | Button | Отправляет сообщение в текущую сессию | `POST /api/session/{id}/messages` |
| **Attach file** | Button | Выбор файла для загрузки (в разработке) | - |
| **AI suggestions**| Button | Запрос подсказок у AI | - |

**Возможные данные:**
- `content`: Любая текстовая строка (сообщение пользователя).

---

## 3. Список сессий (Sessions View)

| Элемент | Тип | Описание / Данные | API Вызов |
| :--- | :--- | :--- | :--- |
| **New Session** | Кнопка | Создает новую сессию | `POST /api/sessions` |
| **Карточка сессии**| Ссылка | Открывает существующую сессию | `GET /api/session/{id}/messages` |

---

## 4. Датасеты (Datasets View)

| Элемент | Тип | Описание / Данные | API Вызов |
| :--- | :--- | :--- | :--- |
| **Поле поиска** | Input | Фильтрация по содержимому, автору или имени файла | - |
| **Вкладки (Tabs)** | Tabs | Переключение типов: All, Text, Images, Audio, Video, Docs | - |
| **Reprocess All** | Кнопка | Запуск полной переиндексации медиа файлов | `POST /etl/reindex-media` |

---

## 5. Синхронизация и импорт (Sync / Import)

| Элемент | Тип | Описание / Данные | API Вызов |
| :--- | :--- | :--- | :--- |
| **Список диалогов** | List | Загрузка списка чатов Telegram для выбора | `GET /etl/sources/telegram/dialogs` |
| **Sync Selected** | Кнопка | Запуск импорта выбранных чатов | `POST /etl/jobs/telegram` |
| **Analyze Chats** | Кнопка | Запуск анализа (описания + теги) для выбранных чатов | `POST /etl/analyze-chats` |
| **Job Status** | Polling | Отслеживание прогресса задачи | `GET /etl/jobs/{job_id}` |

---

## 6. Социальный граф (Network View)

| Элемент | Тип | Описание / Данные | API Вызов |
| :--- | :--- | :--- | :--- |
| **Cluster Filter**| Select | Выбор конкретного кластера пользователей для отображения | - |
| **Build** | Кнопка | Сборка графа (использует кеш, если есть) | `POST /etl/graph/build?force_rebuild=false` |
| **Rebuild** | Кнопка | Полная пересборка графа с нуля | `POST /etl/graph/build?force_rebuild=true` |

---

## 7. Конфигурация Pipeline (Pipelines View)

| Элемент | Тип | Описание / Данные | API Вызов |
| :--- | :--- | :--- | :--- |
| **Vision Provider**| Select | Выбор провайдера: "local" или "ollama" | `POST /api/config` |
| **Text Model** | Input | ID модели HuggingFace (только чтение) | - |
| **Save Changes** | Кнопка | Сохраняет текущие настройки | `POST /api/config` |

---

## 8. Настройки (Settings View)

*Примечание: В текущей версии UI элементы в Settings являются демонстрационными и используют статические данные.*

| Элемент | Тип | Описание |
| :--- | :--- | :--- |
| **Default Model** | Select | Выбор модели по умолчанию (gpt-4o, etc.) |
| **Temperature** | Input | Число от 0 до 1 (Default: 0.7) |
| **Auto-validate** | Switch | Вкл/Выкл автоматическую валидацию сущностей |
| **Stream Results** | Switch | Вкл/Выкл потоковый вывод ответов |
