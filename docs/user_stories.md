# User Stories и Цепочки Вызовов

В этом разделе описаны типичные сценарии использования Summagram и то, как взаимодействуют сервисы под капотом.

## 1. Синхронизация нового чата

**Сценарий:** Юзер заходит в настройки, видит список своих Telegram-чатов и нажимает "Start Sync" для выбранного канала.

**Цепочка вызовов:**
1.  **Frontend**: Вызывает `POST etl/jobs/telegram` с параметрами чата.
2.  **ETL**: `JobManager` ставит задачу в очередь.
3.  **ETL (Worker)**: `TelegramSource.process_chats` начинает скачивание сообщений.
4.  **ETL**: Каждое сообщение сохраняется в SQLite (`raw_documents`).
5.  **ETL**: Вызывается `Indexer.update_index`, который:
    *   Отправляет текст в **Backend** (`/v1/embeddings`).
    *   Получает векторы и сохраняет их в **ChromaDB**.
6.  **Frontend**: Каждые несколько секунд вызывает `GET etl/jobs/{job_id}` для обновления прогресс-бара.

## 2. Вопрос к истории чатов (RAG)

**Сценарий:** Юзер спрашивает в чате: "О чем мы договаривались по проекту вчера?".

**Цепочка вызовов:**
1.  **Frontend**: Отправляет сообщение в **Backend** (`POST /session/{id}/messages`).
2.  **Backend**: Сохраняет сообщение пользователя в БД.
3.  **Backend**: Вызывает **ETL** (через LlamaIndex `ChatEngine`):
    *   **ETL**: Делает семантический поиск в **ChromaDB** по вопросу.
    *   **ETL**: Извлекает релевантные куски текста (контекст).
4.  **Backend**: Формирует финальный промпт (Системный промпт + Контекст + Вопрос).
5.  **Backend**: Отправляет промпт в LLM (OpenRouter/Ollama).
6.  **Backend**: Стримит ответ обратно во **Frontend**.

## 3. Анализ видео-сообщения

**Сценарий:** Юзер находит в "Datasets" видео-сообщение и нажимает "Analyze".

**Цепочка вызовов:**
1.  **Frontend**: Вызывает `POST backend/analyze-video`.
2.  **Backend**: `video_utils.extract_frames` нарезает видео на ключевые кадры.
3.  **Backend**: `LocalInferenceService.analyze_image` вызывается для каждого кадра (используется SmolVLM).
4.  **Backend**: Результаты суммируются и возвращаются во **Frontend**.
5.  **Frontend**: Отображает текстовое описание происходящего в видео.
